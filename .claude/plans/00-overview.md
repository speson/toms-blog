# Passive Income Automation Project - Overview

## Project Summary

2가지 자동화 수익 프로젝트를 구축하는 계획.

| 프로젝트          | 상태           | 우선순위 |
| ----------------- | -------------- | -------- |
| 기술 블로그       | **MVP 진행중** | 1        |
| Suno AI 음악 채널 | 보류           | 2        |

---

## Tech Stack (확정)

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX (로컬 파일)
- **Hosting**: Vercel (무료 티어)
- **Domain**: `toms-blog.vercel.app` (무료)
- **Design**: Dark Minimal (Vercel/Toss 스타일)

## Key Decisions

1. **Next.js 단일 스택** - Python 대신 선택 (운영 복잡도 감소)
2. **블로그 먼저** - 음악 파이프라인은 나중에 결정
3. **완전 무료** - Vercel 무료 티어 + vercel.app 도메인
4. **Semi-Manual 워크플로우** - Claude API 대신 직접 검수 후 발행
5. **다크 모드 기본** - Vercel/Toss 스타일 (모던 SaaS 디자인)
