언어 학습 앱 [Praktika](https://praktika.ai)가 GPT-4.1과 GPT-5.2를 활용한 **멀티 에이전트 튜터링 시스템**을 공개했습니다. 단순한 챗봇이 아닌, 실제 인간 튜터처럼 학습자의 상황에 맞춰 실시간으로 수업을 조정하는 AI입니다.

## 창업자들의 공통 경험

Praktika의 세 창업자 Adam Turaev, Anton Marin, Ilya Chernyakov는 모두 이민자 가정 출신입니다. 새로운 나라에서 영어는 학교, 직장, 일상생활 모든 곳에서 필수였습니다.

> "영어 학습은 단순한 소통이 아니었습니다. 국제적인 일과 커리어 성장의 문을 열어주었죠." - Adam Turaev

문제는 기존 언어 교육이었습니다. 수년간 공부해도 읽기와 쓰기는 유창해지지만, **실제 대화에서는 자신감이 없었습니다**. 교실 학습과 실전 유창성 사이의 간극이 너무 컸습니다.

## 멀티 에이전트 아키텍처

Praktika는 세 개의 전문화된 에이전트가 협력하는 구조입니다:

### 1. Lesson Agent (GPT-5.2)

학습자와 직접 대화하는 메인 튜터입니다. 튜터 성격, 수업 맥락, 학습자 목표, 최근 대화를 종합해서 자연스럽고 즉흥적인 수업을 진행합니다.

### 2. Student Progress Agent (GPT-5.2)

백그라운드에서 계속 실행되며 학습자의 언어 수행을 추적합니다:

- 유창성
- 정확성
- 어휘 사용
- 반복되는 실수

이 데이터가 Lesson Agent의 실시간 행동과 장기 학습 전략에 모두 반영됩니다.

### 3. Learning Planning Agent (GPT-5 Pro)

장기적인 학습 진행을 설계합니다. Student Progress Agent의 인사이트를 바탕으로:

- 다음에 무엇을 배울지
- 어떤 순서로 스킬을 쌓을지
- 어떤 활동이 가장 효과적일지

를 결정합니다.

## 실시간 메모리 시스템

세 에이전트 모두 **지속적인 메모리 레이어**를 공유합니다. 학습자 목표, 선호도, 과거 실수를 저장하죠.

핵심은 타이밍입니다. 컨텍스트를 미리 로드하는 게 아니라, **학습자가 말한 직후에 메모리를 검색**합니다. 그래서 응답이 가장 관련성 높은 최신 신호에 기반합니다.

> "학습자가 지금 실수하면, 튜터는 어제 실수가 아니라 *방금 그 실수*에 반응합니다. 이 미묘한 타이밍 차이가 로봇 같은 느낌 대신 주의 깊은 느낌을 줍니다." - Adam Turaev

## 비원어민 음성 인식

언어 학습자는 머뭇거리고, 문장을 다시 시작하고, 발음이 불완전합니다. Praktika는 **Transcription API**를 사용해 파편화되고 억양 있는 비원어민 음성을 기존 시스템보다 안정적으로 처리합니다.

- **수개월 만에 매출 2배**

## 현재 규모

Praktika는 단순히 언어를 가르치는 게 아닙니다. 사람들이 실제 세계에서 자신감 있게 언어를 사용할 수 있도록 돕는 AI를 만들고 있습니다.
