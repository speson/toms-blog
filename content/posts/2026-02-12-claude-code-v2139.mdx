안녕하세요, Tom입니다.

**Claude Code v2.1.39**가 나왔어요! 이번에는 새로운 기능 추가보다는 **안정성과 성능 개선**에 집중한 릴리스입니다. 특히 터미널 렌더링 성능이 개선됐는데, 긴 세션에서 체감이 됩니다.

## 변경사항

### 터미널 렌더링 성능 개선 🚀

긴 대화에서 터미널이 느려지는 현상이 개선됐어요. 특히 코드 블록이 많은 응답에서 차이가 느껴집니다.

💡 **제가 느낀 점:** 이전 버전에서는 긴 코드를 출력할 때 스크롤이 버벅거리는 느낌이 있었는데, 확실히 부드러워졌어요.

### Fatal Error 표시 수정 🐛

치명적인 에러가 발생했을 때 화면에 표시되지 않고 **조용히 삼켜지던 문제**가 수정됐어요.

⚠️ **이전 문제:** 뭔가 잘못됐는데 에러 메시지가 안 보여서 원인을 모르는 상황이 있었습니다. 이제는 제대로 표시됩니다.

### 세션 종료 후 프로세스 행 수정

세션을 닫았는데 **프로세스가 종료되지 않고 걸려있던 문제**가 해결됐어요. 터미널을 강제로 닫아야 했던 경험이 있으시다면 반가운 소식입니다.

### 터미널 화면 경계 글자 손실 수정

화면 경계에서 **글자가 잘리거나 사라지던 문제**가 수정됐어요. 특히 터미널 폭이 좁을 때 발생하던 문제입니다.

### Verbose 트랜스크립트 빈 줄 수정

`--verbose` 모드에서 트랜스크립트에 **빈 줄이 나타나던 문제**가 수정됐습니다.

## 업데이트 방법

```bash
claude --update
```

## 총평

v2.1.39는 **안정성 릴리스**예요. 화려한 새 기능은 없지만, 매일 쓰는 도구의 기반을 탄탄하게 만드는 종류의 업데이트입니다.

특히 fatal error가 삼켜지던 문제와 세션 종료 후 프로세스 행 문제는 실제로 겪어본 분들이 많을 텐데, 이번에 확실히 해결된 것 같아요. 업데이트 추천드립니다!
