안녕하세요, Tom입니다.

OpenAI가 **GPT-5.3-Codex-Spark**를 발표했어요. 이름에 "Spark"가 붙은 이유가 있습니다 — **실시간 코딩**에 최적화된 모델이에요. 초당 1,000토큰 이상, 기존 대비 **15배 빠른 코드 생성**이 가능하다고 합니다.

## 무엇이 특별한가요?

### 실시간 코딩이라는 새로운 경험 ⚡

기존 코딩 AI는 프롬프트를 주고 기다리는 방식이었잖아요. Codex-Spark는 **즉각적인 피드백**을 목표로 해요:

- 초당 **1,000+ 토큰** 생성
- 코드 수정, 로직 변경, UI 조정이 **거의 실시간**
- GPT-5.3-Codex 대비 **15배 빠른 생성 속도**

💡 **체감 차이:** 일반 Codex가 "요청 → 대기 → 결과"라면, Spark는 "타이핑하면서 바로 결과가 나오는" 느낌을 목표로 하고 있어요.

### NVIDIA 없이 돌아가는 첫 GPT 모델 🔧

이게 정말 흥미로운 점인데요, Codex-Spark는 **Cerebras의 Wafer Scale Engine 3** 위에서 돌아갑니다.

🎯 **왜 중요한가:**

- OpenAI 역사상 **NVIDIA GPU 없이 돌아가는 첫 GPT 모델**
- Cerebras와의 파트너십 — 초저지연 하드웨어에 최적화
- AI 칩 시장의 다변화 신호

### 128k 컨텍스트 윈도우

현재 텍스트 전용이지만, **128k 컨텍스트**를 지원해서 대규모 코드베이스도 다룰 수 있어요.

## 성능은 어떤가요?

SWE-Bench Pro, Terminal-Bench 2.0 등 코딩 벤치마크에서 강력한 성능을 보여준다고 해요. 단순 코드 생성을 넘어 에이전트 역할도 가능합니다:

- 디버깅
- 배포 관리
- 모니터링
- PRD(제품 요구사항 문서) 작성

🤔 **재미있는 사실:** GPT-5.3-Codex의 초기 버전이 자기 자신의 개발 과정에서 디버깅과 테스트를 도왔다고 해요. AI가 AI를 만드는 시대가 정말 와버렸네요.

## 어디서 쓸 수 있나요?

| 채널         | 대상                      |
| ------------ | ------------------------- |
| Codex 앱     | ChatGPT **Pro** 사용자    |
| CLI          | ChatGPT **Pro** 사용자    |
| VS Code 확장 | ChatGPT **Pro** 사용자    |
| API          | 일부 디자인 파트너 (예정) |

⚠️ **현재 제한:** 리서치 프리뷰 단계라 ChatGPT Pro 구독자만 접근 가능하고, 텍스트 전용이에요.

## 총평

GPT-5.3-Codex-Spark는 코딩 AI의 **패러다임 전환**을 시도하는 모델이에요. "더 똑똑하게"가 아니라 "더 빠르게" — 실시간 인터랙션이라는 새로운 UX를 제안합니다.

💰 **현실적 관점:** 현재 Pro 전용이라 월 $200 구독이 필요해요. 비용 대비 가치는 실제로 써봐야 판단할 수 있을 것 같습니다.

Cerebras 하드웨어 채택은 AI 인프라 측면에서도 의미가 큰 뉴스예요. NVIDIA 독점 체제에 균열이 생기고 있다는 신호이기도 합니다.
