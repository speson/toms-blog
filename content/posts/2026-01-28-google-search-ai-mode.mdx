Google이 Search의 새로운 비전을 발표했습니다. **"Just ask anything"** - 아무리 길고 복잡한 질문이라도 필요한 답을 찾을 수 있는 매끄러운 대화형 검색 경험입니다.

## 두 가지 주요 업그레이드

### 1. Gemini 3가 AI Overviews 기본 모델로

전 세계적으로 **Gemini 3**가 AI Overviews의 기본 모델이 됩니다. 도움이 필요한 질문에 대해 검색 결과 페이지에서 바로 최고 수준의 AI 응답을 받을 수 있습니다.

### 2. AI Overview에서 바로 대화 시작

이제 **AI Overview에서 직접 후속 질문**을 할 수 있습니다. AI Mode로 자연스럽게 전환되어 대화형 검색이 가능해집니다.

## 새로운 검색 경험

Google의 테스트 결과, 사용자들은 **자연스럽게 대화로 이어지는 경험**을 선호했습니다. AI Overview의 맥락을 유지하면서 후속 질문을 하면 검색이 훨씬 유용해집니다.

### 기존 vs 새로운 흐름

**기존**:

```
검색 → 결과 확인 → 새 검색어 입력 → 결과 확인 → ...
```

**새로운**:

```
검색 → AI Overview 확인 → 바로 후속 질문 → 대화 계속 → ...
```

## 하나의 유연한 경험

- **빠른 정보**: 스포츠 점수, 날씨 같은 단순 질문은 간단한 결과로
- **깊은 탐색**: 복잡한 질문은 대화형 AI Mode로 심층 탐구

필요에 따라 빠른 스냅샷과 깊은 대화 사이를 자연스럽게 오갈 수 있습니다.

## 사용 방법

1. 평소처럼 Google에서 검색
2. AI Overview가 표시되면 확인
3. 추가 질문이 있으면 **바로 입력**
4. AI Mode로 자동 전환되어 대화 계속

## 글로벌 출시

이 기능은 **모바일**에서 **전 세계**에 출시됩니다.

## 핵심 메시지

>
> - Robby Stein, VP of Product, Google Search

검색의 미래는 **정보 탐색, 접근, 이해를 더 쉽게** 만드는 것입니다. 이번 업데이트로 Google Search가 그 비전에 한 걸음 더 가까워졌습니다.
