안녕하세요, Tom입니다.

**Oh My OpenCode** v3.1.4가 릴리스되었습니다! 이번 버전은 Provider Cache 관련 UX 개선과 npm 글로벌 설치 환경에서의 버그 수정이 주를 이룹니다.

## 새로운 기능

### Provider Cache Warning Toast

Provider Cache가 누락된 경우 **토스트 알림**을 표시합니다. 모델 해석(resolution) 문제가 발생했을 때 사용자가 원인을 파악하기 쉬워졌습니다.

💡 **이전에는:** Provider Cache가 없으면 조용히 실패하거나 예상치 못한 동작이 발생했어요.

🎯 **이제는:** 명확한 알림으로 문제를 즉시 인지할 수 있습니다.

## 버그 수정

### npm 글로벌 설치 버전 감지

`npm install -g oh-my-opencode`로 설치한 경우 버전 감지가 제대로 작동하지 않던 문제가 수정되었습니다.

⚠️ **이 문제 겪으신 분들:** 이제 정상적으로 버전이 감지됩니다.

### Model Resolver 개선

Provider Cache가 없을 때 fallback 체인을 건너뛰도록 변경되어 불필요한 에러가 방지됩니다.

### Config Schema 업데이트

`BrowserAutomationProviderSchema`에 `'dev-browser'` 옵션이 추가되었습니다.

## CI & 테스트 개선

- 병렬 실행 안정성을 위한 테스트 격리 개선
- CI 테스트 타임아웃을 설정 가능하도록 수정
- delegate-task 테스트의 mock 처리 개선

## 업데이트 방법

```bash
# Homebrew (macOS)
brew upgrade oh-my-opencode

# npm
npm update -g oh-my-opencode

# 직접 설치
curl -fsSL https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/main/install.sh | bash
```

## 제 생각

v3.1.4는 **안정성 개선**에 초점이 맞춰진 마이너 업데이트입니다. Provider Cache 관련 UX 개선이 특히 유용합니다.

🎯 **Oh My OpenCode를 쓰고 계시다면:** 이번 업데이트를 권장합니다. npm 글로벌 설치 버그가 수정되어 더 안정적으로 사용할 수 있어요.
