안녕하세요, Tom입니다.

이번에 소개할 **OpenCode**는 제가 최근 한 달간 메인 코딩 도구로 쓰고 있는 AI 에이전트입니다. Claude Code가 유료화되면서 대안을 찾고 있었는데, 이 친구가 꽤 괜찮더라고요.

## OpenCode란? (그리고 제가 선택한 이유)

OpenCode는 터미널 기반 **오픈소스 AI 코딩 에이전트**입니다. Claude Code와 거의 비슷한 기능을 제공하면서도 **완전히 무료**라는 점이 가장 큰 장점입니다.

GitHub 스타가 8만 개 이상인데, 이 정도면 신뢰할 수 있는 프로젝트라고 봅니다. 커뮤니티도 활발하고 업데이트도 자주 나와요.

💡 **제가 OpenCode를 선택한 이유:**

- Claude Code는 월 $20인데, OpenCode는 공짜
- Claude, GPT, Gemini 중에서 AI 모델을 골라 쓸 수 있음
- LSP 지원이 기본 내장되어 있어서 코드 분석이 정확함

## Claude Code와 비교

| 항목     | OpenCode               | Claude Code |
| -------- | ---------------------- | ----------- |
| 가격     | **무료**               | 월 $20      |
| AI 모델  | Claude, GPT, Gemini 등 | Claude 전용 |
| 오픈소스 | ✅ MIT 라이선스        | ❌ 비공개   |
| LSP 지원 | ✅ 기본 내장           | ⚠️ 제한적   |

🎯 **Claude Code가 더 나은 점:**

- UI/UX가 더 세련됨
- Anthropic의 공식 지원
- 더 안정적인 동작

**OpenCode가 더 나은 점:**

- **무료** (이게 제일 큼)
- AI 모델 선택의 자유
- 오픈소스라 커스터마이징 가능

## 설치는 정말 쉬워요

한 줄이면 끝입니다:

```bash
curl -fsSL https://opencode.ai/install | bash
```

Homebrew도 지원합니다:

```bash
brew install anomalyco/tap/opencode
```

⚠️ **주의:** Node.js 18+가 필요합니다. 설치 전에 확인하세요.

## 실제 사용해보니

### 좋았던 점

**1. AI 모델 선택의 자유**

Claude Code는 Claude만 쓸 수 있는데, OpenCode는 Claude, GPT, Gemini 중에서 골라 쓸 수 있습니다. 저는 Claude 3.5 Sonnet을 주로 쓰는데, 가끔 GPT-4o로 바꿔서 써보기도 해요.

🎯 **팁:** Claude 3.5 Sonnet이 코딩에서는 여전히 최고입니다. GPT-4o는 빠르지만 코드 품질은 Sonnet이 더 나아요.

**2. LSP 지원이 기본 내장**

TypeScript 프로젝트에서 타입 에러를 찾아내는 능력이 인상적이었어요. Claude Code보다 코드 분석이 더 정확한 느낌입니다.

**3. 확장성**

오픈소스라서 필요하면 직접 수정할 수 있습니다. 저는 프롬프트 템플릿을 약간 커스터마이징해서 쓰고 있어요.

### 아쉬웠던 점

**1. UI가 좀 투박해요**

Claude Code보다 TUI가 덜 세련된 느낌입니다. 기능은 다 있는데, 눈으로 보기에는 Claude Code가 더 예쁘더라고요.

**2. 초기 설정이 필요함**

Claude Code는 바로 쓸 수 있는데, OpenCode는 API 키 설정 같은 초기 설정이 좀 필요합니다. 한 번만 하면 되지만, 귀찮긴 해요.

**3. 공식 지원이 없음**

문제가 생기면 커뮤니티에 의존해야 합니다. Claude Code는 공식 지원이 있어서 더 안심되죠.

## 누구에게 추천하나요?

**추천:**

- Claude Code 비용이 부담스러운 개발자
- 오픈소스 도구를 선호하는 분
- 여러 AI 모델을 테스트해보고 싶은 분

**비추천:**

- GUI를 선호하는 분 (터미널 기반입니다)
- 설정 없이 바로 쓰고 싶은 분
- 공식 지원이 필요한 분

## 총평

OpenCode는 Claude Code의 **무료 대안으로서 충분히 경쟁력** 있습니다. UI는 덜 세련됐지만 기능은 거의 비슷하고, 오히려 AI 모델 선택의 자유도가 더 높아요.

💰 **비용 절약 팁:** Claude Code 월 $20 vs OpenCode $0. 1년이면 **$240 절약**됩니다. 성능 차이는 거의 없어요.

물론 Claude Code의 공식 지원과 안정성이 필요하신 분들은 월 $20을 내고 쓰는 것도 충분히 가치 있습니다. 하지만 비용을 절약하고 싶으신 분들은 OpenCode를 강력히 추천드립니다.
