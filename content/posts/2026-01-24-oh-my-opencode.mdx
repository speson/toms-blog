오늘은 **Oh My OpenCode**라는 흥미로운 프로젝트를 소개할게요. OpenCode 위에서 동작하는 오케스트레이션 레이어인데, GitHub 스타 22K를 받을 정도로 인기가 많더라고요.

## Oh My OpenCode가 뭔가요?

**Oh My OpenCode**는 [OpenCode](/posts/2026-01-24-opencode-introduction) 위에서 동작하는 **오케스트레이션 레이어**입니다. OpenCode가 리눅스라면, Oh My OpenCode는 Ubuntu와 같은 존재라고 생각하면 됩니다.

💡 **쉽게 말하면:** OpenCode를 더 강력하게 만들어주는 플러그인 모음집입니다. 설치만 하면 바로 사용할 수 있는 **배터리 포함(Batteries-Included)** 환경을 제공합니다.

### 주요 수치

| 지표         | 수치        |
| ------------ | ----------- |
| GitHub Stars | **22,600+** |
| Forks        | **1,600+**  |
| Contributors | **68+**     |

🎯 **22K 스타는 무시할 수 없는 수치입니다.** 이 정도면 검증된 프로젝트라고 봐도 됩니다.

## OpenCode vs Oh My OpenCode

| 항목          | OpenCode                  | Oh My OpenCode                          |
| ------------- | ------------------------- | --------------------------------------- |
| 역할          | AI 코딩 에이전트 (베이스) | 오케스트레이션 레이어 (플러그인)        |
| 에이전트      | Build, Plan               | Sisyphus, Oracle, Librarian, Explore 등 |
| 멀티 에이전트 | 단일 에이전트             | 병렬 멀티 에이전트 지원                 |
| LSP/AST       | 기본 지원                 | 고급 리팩토링 도구 포함                 |
| MCP           | 수동 설정                 | Exa, Context7, Grep.app 내장            |

⚠️ **Claude Code 호환성이 특징입니다.** Oh My OpenCode는 Claude Code와 완벽하게 호환됩니다. Claude Code를 쓰시는 분들도 Oh My OpenCode의 기능을 활용할 수 있어요.

## 설치 방법

### 가장 쉬운 방법: LLM 에이전트에게 설치 요청

Claude Code나 Cursor에게 이렇게 말해보세요:

```
Install and configure oh-my-opencode by following the instructions here:
https://github.com/code-yeongyu/oh-my-opencode
```

AI가 알아서 설치해줍니다. 정말 편하죠?

### 수동 설치

```bash
# npm으로 설치
npm install -g oh-my-opencode

# 또는
npx oh-my-opencode install
```

## 제가 주목한 기능

**1. 멀티 에이전트 병렬 처리**

단일 에이전트가 아니라 여러 에이전트가 동시에 작업할 수 있습니다. 예를 들어:

- 한 에이전트는 코드 검색
- 다른 에이전트는 문서 작성
- 또 다른 에이전트는 테스트 실행

🎯 **이게 진짜 강력합니다.** 시간이 크게 단축됩니다.

**2. 고급 LSP/AST 도구**

기본 OpenCode도 LSP를 지원하지만, Oh My OpenCode는 더 고급 리팩토링 도구를 제공합니다. 대규모 코드베이스에서 특히 유용합니다.

**3. MCP 내장**

Exa, Context7, Grep.app 등이 기본으로 내장되어 있습니다. 별도로 설정할 필요가 없어요.

## 제 생각

Oh My OpenCode는 **OpenCode의 진화형**이라고 볼 수 있습니다. OpenCode가 좋은 기반이라면, Oh My OpenCode는 그 위에 실용적인 기능들을 쌓아올린 거죠.

💰 **Claude Code 사용자라면:** Oh My OpenCode의 기능을 Claude Code에서도 쓸 수 있다는 점이 매력적입니다. Claude Code 월 $20 내고 쓰시는 분들은 Oh My OpenCode로 기능을 확장해보세요.

**OpenCode 사용자라면:** 거의 필수적으로 설치해야 하는 레이어입니다. 기본 기능이 훨씬 풍부해집니다.

## 마무리

Oh My OpenCode는 OpenCode 생태계의 중요한 프로젝트입니다. 22K 스타를 받을 정도로 많은 개발자들이 인정한 도구이죠.

여러분도 OpenCode나 Claude Code를 쓰고 계시다면, Oh My OpenCode를 한 번 설치해보세요. 생각보다 많은 것이 달라집니다.
